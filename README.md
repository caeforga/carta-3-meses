# Carta para Yennifer

Página romántica con acceso por fecha y carta con efecto de escritura. Lista para GitHub Pages.

## Personalizar

Edita `config.js`:

- `passwordDate`: fecha en que se hicieron novios (`DD/MM/AAAA`)
- `letterText`: el texto de la carta
- `signature`: la firma

## Probar en local

Abre `index.html` en el navegador, o usa un servidor local:

```bash
npx serve .
```

## Publicar en GitHub Pages

1. Crea un repositorio en GitHub y sube estos archivos.
2. En el repo: **Settings → Pages**.
3. En **Source**, elige **Deploy from a branch**.
4. Branch: `main` (o `master`), carpeta `/ (root)`.
5. Guarda y espera 1–2 minutos.
